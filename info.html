WebCam Spyware Security Project

This is an open-source project designed to provide users with tools to manage and secure their webcam. It offers features to check webcam status, monitor activity through logs, and control webcam access. The project is currently under development as part of an internship.

Key Features

    [cite_start]
  • View Logs: Check a detailed log of when the webcam was enabled or disabled[cite: 2].
  • Check Status: Determine the current operational status of your webcam.
  • Change Password: Securely update the password required for critical actions.
  • Enable/Disable Camera: Control the accessibility of your webcam to prevent unauthorized use. (Note: Actual hardware control for enabling/disabling is a planned enhancement.)

How to Run the Project

To get started with the WebCam Spyware Security application, follow these steps:

  1. Clone the Repository: Download the project files to your local machine.
  2. Install Dependencies: Ensure you have Python and any required modules installed.
  3. Execute the Application: Open a terminal or command prompt, navigate to the project directory, and run: python interface.py.
  4. Default Password: When prompted to enable or disable the camera, use the default password "admin".

Technologies Used

  • Python: The core programming language.
  • Tkinter: For building the graphical user interface.
  • PIL (Pillow): For image processing (e.g., loading the webcam icon).
  • subprocess module: For running system commands (e.g., checking webcam status).
  • hashlib module: For secure password hashing.
