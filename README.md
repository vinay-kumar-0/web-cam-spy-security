# WebCam Spy Security

This project is an open-source initiative designed to help users manage and secure their webcam. It provides essential features to monitor webcam activity, check its status, and control its accessibility to enhance privacy and prevent unauthorized use.

## 🌟 Features

* [cite_start]**View Webcam Logs**: Keep track of when your webcam was enabled or disabled, providing a history of its activity[cite: 2].
* [cite_start]**Check Webcam Status**: Instantly determine the current operational status of your webcam[cite: 1, 2].
* [cite_start]**Secure Password Protection**: Critical actions like enabling or disabling the camera are protected by a password[cite: 1, 2].
* [cite_start]**Change Password**: Update your security password anytime for enhanced protection[cite: 1, 2].
* [cite_start]**Enable/Disable Camera**: Gain control over your webcam, allowing you to turn it on or off as needed (Note: Actual hardware control for enabling/disabling is a planned enhancement in progress). [cite: 1, 2]

## 🚀 Getting Started

Follow these simple steps to get the "WebCam Spy Security" project up and running on your system:

### Prerequisites

* Python 3.x installed on your system.
* Required Python modules: `tkinter`, `Pillow (PIL)`, `subprocess`, `hashlib`.

### Installation and Execution

1.  **Clone the Repository**:
    ```bash
    git clone [https://github.com/your_username/web-cam-spy-security.git](https://github.com/your_username/web-cam-spy-security.git)
    cd web-cam-spy-security
    ```
    (Note: Replace `https://github.com/your_username/web-cam-spy-security.git` with the actual repository URL.)

2.  **Install Required Modules**:
    ```bash
    pip install Pillow
    ```
    (Other modules like `tkinter`, `subprocess`, `hashlib` are usually part of standard Python distributions.)

3.  **Run the Application**:
    Navigate to the directory where you've cloned the files and execute the main interface script:
    ```bash
    python interface.py
    ```

4.  **Default Password**:
    [cite_start]When prompted to enable or disable the camera, the **default password is `admin`**[cite: 1, 2]. You can change this password later using the "Change Password" option in the application.
